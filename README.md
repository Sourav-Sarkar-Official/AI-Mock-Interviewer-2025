# AI Mock Interviewer — Web App

Our product is an AI-powered mock interviewer that simulates realistic technical and behavioral interviews. It generates tailored questions, provides instant feedback, and tracks progress with detailed performance analytics to help users improve interview skills and confidence. Designed for developers and job-seekers, it supports multiple roles, customizable difficulty based on working experience, and real-time problem-solving environments.

---

## Key Features

- Generate mock interviews from curated question banks
- Save your answers securely to Database
- User authentication with basic profile customization
- Interview profile management and session history
- Clean, accessible and user friendly UI

## Live Demo

[https://ai-mock-interviewer-app.web.app/](https://ai-mock-interviewer-app.web.app/)

## Tech Stack

- Framework: React + Vite
- Language: TypeScript
- Styling: Tailwind CSS
- Auth: Clerk
- Backend / Hosting: Firebase

## Getting Started — Local Development
```bash

```

Run the dev server:

